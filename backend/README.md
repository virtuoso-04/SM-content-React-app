# Backend API

FastAPI backend with multi-model AI routing.

## Setup

```bash
pip install -r requirements.txt
# Add GEMINI_API_KEY to .env
python main.py
```

## Endpoints

- `/health` - Health check
- `/api/summarize` - Summarize text
- `/api/generate-ideas` - Generate ideas
- `/api/refine-content` - Refine content
- `/api/chat` - Chat with AI
- `/api/smart-route` - Multi-model routing
- `/docs` - Full API documentation

## Start

```bash
./start.sh
```

Server: http://localhost:8000
